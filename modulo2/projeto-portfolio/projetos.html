Projeto BlackJack

O objetivo deste projeto é criar um jogo semelhante a Blackjack (também conhecido como "21").

A rodada consiste em sortear 2 cartas para cada jogador, definir a pontuação de cada jogador e informar o vencedor.

Esse projeto foi criado atraves da liguagem Javascript
